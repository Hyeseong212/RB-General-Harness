# fifth acting 작업 히스토리

- 작성일: 2026-05-12

## 사용자 요청

연결 화면 애니메이션 클립 다음으로, 데이터 플로우 차트 샘플을 fifth acting에 추가한다.

## 해석

third acting의 화면상세와 fourth acting의 연결 화면 애니메이션은 “화면이 어떻게 움직이는지”를 보여준다. fifth acting은 “그 화면 액션이 어떤 데이터로 저장되고, 치료사 화면에서 무엇을 조회하는지”를 보여주는 역할로 분리한다.

- 데이터 플로우 차트 SVG 제작

- 엔티티/이벤트/read model JSON 명세 작성
- 설명 문서 작성
- SVG XML 파싱 확인
- JSON 파싱 확인
- Chrome headless 렌더 캡처 확인
- 저장·조회 영역 화살표 정리

- 입력/트리거, API/이벤트 처리, 저장 테이블, 화면 조회/출력 4단 구조로 재정리
