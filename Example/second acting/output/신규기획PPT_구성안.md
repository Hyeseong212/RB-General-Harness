# 신규 기획 PPT 구성안

- 작성일: 2026-05-12
- 단계: second acting
- 대상 PPT: 이승로봇 게임재활 콘텐츠 기획 PPT
- 목적: first acting의 자료조사/브레인스토밍 결과를 바탕으로, 새로운 기획 발표자료의 흐름을 설계한다.
- 정정 메모: 이 PPT는 기존 하네싱 발표자료의 9페이지 이후에 삽입하는 장표가 아니다. 별도의 신규 기획 PPT로 본다.

## PPT의 한 문장 목표

간단한 재활운동을 반복 숙제가 아니라, 목표·반응·보상·기록이 있는 게임운동 콘텐츠로 설계하는 방향을 제안한다.

## 발표 대상

- 내부 기획/개발/디자인/재활 자문 담당자
- 이승로봇 콘텐츠 방향을 검토할 PM
- 이후 UI/UX 또는 프로토타입 제작을 맡을 에이전트/작업자

## 발표 톤

- 제품 효과를 주장하는 세일즈 PPT가 아니다.
- 아이디어를 보여주되, 재활 안전성과 표현 기준을 같이 잡는 기획 PPT다.
- 재미있는 콘텐츠 방향을 제안하지만, "치료된다", "회복된다", "효과가 입증됐다" 같은 단정 표현은 쓰지 않는다.

## 권장 슬라이드 구성

### 1페이지. 운동은 그대로인데, 경험이 달라질 수 있다

**역할**

표지 겸 핵심 컨셉 제시.

**핵심 메시지**

간단한 재활운동을 짧은 미션과 피드백이 있는 게임운동 경험으로 바꾼다.

**시각화 아이디어**

왼쪽에는 단순 반복운동, 오른쪽에는 게임 미션 화면을 대비한다.

```text
팔 뻗기 / 손 쥐기 / 무릎 펴기
→ 미션 / 반응 / 점수 / 기록
```

---

### 2페이지. 기존 재활운동 콘텐츠는 반복의 이유를 보여주기 어렵다

**역할**

문제 정의.

**핵심 메시지**

반복 자체보다, 반복을 계속 이해하고 참여하게 만드는 경험 설계가 필요하다.

**넣을 내용**

- 단순 반복은 지루하게 느껴질 수 있다.
- 환자는 "잘하고 있는지"를 즉시 알기 어렵다.
- 보호자와 치료사는 수행 과정과 중단 이유를 확인하고 싶다.
- 관리자는 콘텐츠 운영 기준과 기록 흐름이 필요하다.

**주의**

실제 환자 행동을 단정하지 않는다. "그럴 수 있다", "필요할 수 있다" 수준으로 표현한다.

---

### 3페이지. 게임화의 핵심은 재미가 아니라 반복 루프를 만드는 것이다

**역할**

기획 원리 제시.

**핵심 메시지**

게임 요소는 운동을 대신하는 것이 아니라, 반복 수행을 이해하기 쉬운 루프로 바꾸는 장치다.

**시각화 아이디어**

큰 순환 다이어그램.

```text
목표 제시
→ 동작 수행
→ 화면 반응
→ 점수/진행률
→ 휴식/안전 체크
→ 기록 저장
→ 다음 미션
```

---

### 4페이지. 기본 운동은 작은 게임 미션으로 바뀐다

**역할**

콘텐츠 예시 제시.

**핵심 메시지**

운동 동작 하나하나는 게임 속 미션, 반응, 기록 단위로 재구성할 수 있다.

**넣을 표**

| 기본 운동 | 게임 미션 | 화면 반응 | 기록 |
| --- | --- | --- | --- |
| 팔 앞으로 뻗기 | 과일 따기 | 목표 지점에 닿으면 빛이 켜짐 | 도달 횟수, 좌우 차이 |
| 손 쥐었다 펴기 | 보석 모으기 | 손 동작마다 아이템 획득 | 반복 횟수, 피로도 |
| 무릎 펴기 | 풍선 터뜨리기 | 다리 움직임에 따라 타일 반응 | 수행 시간, 중단 사유 |
| 체중 이동 | 균형 게이지 맞추기 | 중심이 범위 안에 들어오면 유지 | 유지 시간, 흔들림 |

**주의**

"기능 향상", "회복" 같은 결과 표현은 넣지 않는다.

---

### 5페이지. 환자에게는 오늘의 미션을 보여준다

**역할**

환자 관점.

**핵심 메시지**

환자 화면은 치료 결과보다 "오늘 무엇을 하면 되는지"를 쉽게 보여주는 데 집중한다.

**넣을 내용**

- 오늘의 미션
- 남은 횟수 또는 진행률
- 휴식 안내
- 다시 시도 문구
- 통증/피로 입력

**안전한 문장 후보**

"오늘의 미션을 하나씩 수행합니다."

---

### 6페이지. 치료사에게는 게임 결과가 관찰 기록이 된다

**역할**

치료사 관점.

**핵심 메시지**

게임 결과는 치료사가 참고할 수 있는 수행 기록으로 정리되어야 한다.

**넣을 내용**

- 반복 횟수
- 수행 시간
- 중단 사유
- 좌우 차이
- 통증/피로 메모
- 다음 세션 참고 사항

**안전한 문장 후보**

"수행 기록은 치료사의 판단을 돕는 참고 자료입니다."

---

### 7페이지. 보호자와 관리자는 안전과 운영 흐름을 본다

**역할**

보호자/관리자 관점.

**핵심 메시지**

게임 콘텐츠는 재미뿐 아니라 관찰, 안전 체크, 운영 기준을 함께 가져야 한다.

**넣을 내용**

| 관점 | 필요한 정보 | 콘텐츠 방향 |
| --- | --- | --- |
| 보호자 | 피로, 통증, 변화 관찰 | 상담 참고 메모 |
| 관리자 | 세션 수, 사용 현황, 안전 체크 | 운영 기록 |
| 재활 전문가 | 대상자, 운동 범위, 중단 기준 | 검토 질문 |

---

### 8페이지. 재미있는 문장과 안전한 문장은 따로 검토한다

**역할**

표현 기준 제시.

**핵심 메시지**

게임처럼 보여도 재활 콘텐츠는 안전한 문장, 위험한 문장, 출처, 전문가 질문을 분리해야 한다.

**넣을 내용**

| 구분 | 예시 |
| --- | --- |
| 안전한 문장 | "반복 수행을 더 이해하기 쉬운 형태로 설계합니다." |
| 위험한 문장 | "게임을 하면 재활 효과가 올라갑니다." |
| 출처 | 재활 정의, 반복 훈련 근거, 의료기기/광고 표현 기준 |
| 전문가 질문 | 어떤 운동을 게임화해도 되는가 |

---

### 9페이지. first acting과 second acting은 역할이 다르다

**역할**

하네싱 운영 구조 설명.

**핵심 메시지**

자료조사 에이전트는 PPT를 바로 만드는 사람이 아니라, 다음 에이전트가 믿고 쓸 수 있는 근거와 표현 기준을 정리하는 역할이다.

**넣을 내용**

```text
first acting
게임화 아이디어 / 안전 문장 / 위험 문장 / 출처 / 전문가 질문

second acting
PPT 구조 / 장표 문구 / 시각화 방향 / 발표 흐름
```

---

### 10페이지. 다음 단계는 화면 흐름과 작은 프로토타입이다

**역할**

후속 액션 제안.

**핵심 메시지**

기획 PPT 이후에는 한 가지 운동을 골라 30초짜리 게임 미션으로 실험한다.

**넣을 작은 Task**

1. 팔 뻗기 운동 하나를 선택한다.
2. 30초 미션 화면을 만든다.
3. 성공/실패 대신 도달/휴식/다시 시도 피드백을 쓴다.
4. 환자 화면과 치료사용 기록 화면을 분리한다.
5. 재활 전문가가 중단 기준과 표현을 검토한다.

## 디자인 방향

- 16:9, Pretendard 기준
- 한 장에 하나의 메시지
- 긴 조사문 금지
- 루프, 흐름, 비교, 화면 예시 중심
- 실제 제품 화면처럼 보이면 "기획 예시" 표시
- 색상은 게임적 생동감을 주되 의료/재활 콘텐츠의 신뢰감을 해치지 않게 제한

## 전체 메시지 흐름

```text
문제: 반복운동은 경험 설계가 부족하면 지루해질 수 있다
제안: 운동을 미션·피드백·기록 루프로 바꾼다
예시: 팔 뻗기, 손 쥐기, 무릎 펴기 등을 게임 미션화한다
운영: 환자·치료사·보호자·관리자 관점을 분리한다
안전: 재미와 효과 표현을 분리하고 전문가 검토를 남긴다
다음: 한 운동을 골라 작은 프로토타입으로 검증한다
```
