# second acting 인계메모

- 작성일: 2026-05-12
- 인계 대상: PPT 제작 에이전트, UI/UX 기획 에이전트, PM/재활 전문가
- 작업 범위: 자료조사 기반 정리 완료. 최종 PPT는 제작하지 않음.

## 완료 요약

이승로봇 콘텐츠 기획의 first acting 자료조사를 완료했다. 안전한 방향은 "치료 효과"가 아니라 "반복 훈련, 기록, 관찰, 피드백을 전문가 계획 안에서 보조하는 시스템"이다. 모든 의료 효과, 제품 성능, 인허가, 환자 안전, 개인정보 문장은 WARN 또는 전문가 확인 대상으로 분리했다.

## 방향 보정 메모

PPT의 중심 주제는 단순한 재활 근거 조사가 아니라 "간단한 재활운동을 게임운동 콘텐츠로 승화시키는 기획"이다. 따라서 second acting에서는 법률/근거 설명을 앞세우기보다, "운동 → 미션 → 피드백 → 기록 → 다음 미션"으로 이어지는 게임 루프를 중심에 두고 장표화한다. 안전 표현과 팩트체킹은 주인공이 아니라, 게임화 기획이 과장되지 않도록 받쳐 주는 검증 장치로 사용한다.

## second acting에서 바로 사용할 핵심 메시지

1. "회복을 단정하지 않고, 과정을 기록합니다."
2. "반복 훈련, 관찰, 피드백을 한 흐름으로 연결합니다."
3. "환자에게는 이해하기 쉬운 목표를, 치료사에게는 확인 가능한 기록을 제공합니다."
4. "재활 로봇은 치료를 대신하기보다 전문가의 계획과 관찰을 보조하는 시스템으로 설명해야 합니다."
5. "안전한 콘텐츠는 효과 주장보다 사용 맥락, 기록, 주의사항, 전문가 확인을 먼저 보여줍니다."

## PPT 제작 시 권장 구조

1. 문제 정의: 재활 콘텐츠에서 단정 표현을 피해야 하는 이유
2. 기준: WHO의 재활 정의와 사람 중심 접근
3. 근거: NICE의 반복 과제 훈련 권고
4. 조심할 점: 로봇 보조 재활 근거는 장비·부위·대상별로 다름
5. 콘텐츠 방향: 반복 훈련, 기록, 관찰, 피드백
6. 관점별 정보: 환자, 보호자, 치료사, 관리자
7. 표현 가이드: 쓸 수 있는 문장과 위험한 문장
8. 확인 게이트: 재활 전문가, RA, 개인정보 담당자 검토 필요

## 사용 가능한 안전 문장

- "재활은 개인의 기능과 일상 참여를 돕기 위한 과정입니다."
- "반복 과제 훈련은 뇌졸중 재활에서 다뤄지는 중요한 접근 중 하나입니다."
- "훈련 기록은 치료사가 상태 변화를 확인하고 계획을 조정하는 데 참고 자료가 될 수 있습니다."
- "로봇 보조 시스템은 반복 수행과 피드백을 구조화하는 도구로 설명할 수 있습니다."
- "전문가의 계획과 관찰 안에서 사용되는 보조 시스템이라는 관점을 유지합니다."

## 사용 금지 또는 WARN 문장

- "치료된다."
- "회복된다."
- "효과가 입증됐다."
- "기능이 좋아진다."
- "기존 재활보다 우수하다."
- "전문가 없이 안전하게 사용할 수 있다."
- "치료사를 대체한다."
- "식약처가 효과를 보증했다."
- "AI가 환자 상태에 맞춰 치료를 결정한다."

## WARN/FAIL/BLOCKED

- WARN: 로봇 보조 재활 효과 표현은 질환, 부위, 장비, 프로토콜별 근거가 달라 단정 금지.
- WARN: 이승로봇의 의료기기 인허가 상태를 확인하지 못했으므로 제품 효과/안전성/허가 표현 금지.
- WARN: 운동 기록과 환자 상태 데이터는 개인정보 또는 민감정보 가능성이 있어 데이터 설계 확인 필요.
- BLOCKED: 재활 전문가 확인 전 환자용 최종 문구 확정 불가.
- BLOCKED: RA 확인 전 대외 발표용 의료기기 효능·효과 문구 확정 불가.
