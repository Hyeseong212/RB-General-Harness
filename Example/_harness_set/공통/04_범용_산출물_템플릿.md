# 범용 산출물 템플릿

## 1. Task 브리프

```text
# Task 이름

## 목적

## 입력 자료

## 담당 역할

## 출력 산출물

## 검증 기준

## 사람 승인 필요 항목

## 다음 작업
```

## 2. 회의록 정리

```text
# 회의록 정리

## 회의 목적

## 결정사항

## 액션 아이템
| 담당 | 할 일 | 기한 | 확인 기준 |
|---|---|---|---|

## 보류사항

## 리스크

## 다음 회의 전 준비물
```

## 3. 요구사항 정리

```text
# 요구사항 정리

| ID | 요구사항 | 근거 | 담당 | 산출물 | 검증 기준 | 상태 |
|---|---|---|---|---|---|---|
```
