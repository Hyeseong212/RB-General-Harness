# 재활전문가 확인질문

- 확인일: 2026-05-12
- 목적: 콘텐츠 문구와 UI/PPT 메시지를 확정하기 전 재활 전문가에게 확인할 질문
- 원칙: 아래 질문은 에이전트 결과만으로 확정하지 않는다.

## 대상자와 적용 범위

1. 이승로봇 콘텐츠의 주 대상 질환은 무엇인가요? 예: 뇌졸중, 척수손상, 파킨슨병, 근골격계 질환, 고령자 기능 저하.
2. 콘텐츠에서 상지, 하지, 보행, 균형, 손 기능 중 어떤 영역을 우선 설명해야 하나요?
3. 로봇/보조 시스템을 사용할 수 있는 대상자 기준과 제외 기준은 무엇인가요?
4. 급성기, 아급성기, 만성기 중 어떤 재활 단계에 맞춘 콘텐츠인가요?
5. 환자용 문구에서 "훈련", "재활", "운동", "치료" 중 어떤 용어가 가장 적절한가요?

## 안전과 중단 기준

1. 훈련 중 즉시 중단해야 하는 신호는 무엇인가요? 예: 통증, 어지러움, 과도한 피로, 호흡곤란, 낙상 위험.
2. 보호자가 관찰해 기록해도 되는 항목과 치료사가 직접 평가해야 하는 항목은 어떻게 구분하나요?
3. 가정/기관/병원 환경별로 반드시 달라져야 하는 안전 문구가 있나요?
4. 로봇 착용 또는 보조 중 피부 압박, 관절 가동범위, 균형 문제 등 어떤 주의사항을 콘텐츠에 넣어야 하나요?
5. "안전하게 반복 훈련"이라는 표현은 환자용 콘텐츠에 사용할 수 있나요, 아니면 조건부 표현으로 바꿔야 하나요?

## 반복 훈련과 기록

1. "반복 훈련"을 핵심 메시지로 삼는 것이 이승로봇의 실제 사용 시나리오와 맞나요?
2. 반복 횟수, 시간, 강도, 성공률 같은 지표 중 환자에게 보여줘도 되는 지표는 무엇인가요?
3. 치료사에게는 어떤 기록이 가장 의미 있나요? 예: 세션 시간, 반복 수, 보조 수준, 피로도, 통증, 중단 사유.
4. 보호자에게 보여주면 오해가 생길 수 있는 지표가 있나요?
5. 기록을 "개선"으로 해석하지 않고 "변화 추이"로 보여주는 방식이 적절한가요?

## 피드백과 UI/UX

1. 환자에게 즉시 피드백을 줄 때 어떤 표현이 안전한가요? 예: "좋아요", "목표에 가까워졌어요", "전문가와 확인하세요".
2. 피드백이 환자에게 무리한 반복을 유도하지 않도록 어떤 제한 문구가 필요한가요?
3. 치료사용 화면에서 "권장", "주의", "중단" 같은 단어를 사용할 수 있나요?
4. 환자용 화면과 치료사용 화면의 메시지 톤을 어떻게 다르게 잡아야 하나요?
5. 로봇이 제공하는 수치와 치료사의 임상 평가를 함께 보여줄 때 어떤 설명이 필요하나요?

## 표현 검토

1. "재활을 돕습니다"는 안전한 표현인가요?
2. "회복을 지원합니다"는 제품 효과처럼 보일 위험이 있나요?
3. "반복 훈련으로 기능 향상을 돕습니다"는 사용할 수 있나요, 아니면 "반복 수행을 구조화합니다"가 더 적절한가요?
4. "치료사의 판단을 보조합니다"라는 표현은 적절한가요?
5. "환자의 상태 변화를 기록합니다"와 "환자의 회복 변화를 기록합니다" 중 어떤 표현이 안전한가요?

## 출처 기반 확인 필요

| 확인 대상 | 현재 근거 | 전문가에게 확인할 이유 |
| --- | --- | --- |
| 반복 과제 훈련 | NICE NG236, 2023-10-18, https://www.nice.org.uk/guidance/ng236/chapter/Recommendations | 대상 질환과 부위가 이승로봇 콘텐츠와 맞는지 확인 필요 |
| 로봇 보조 상지 훈련 | Cochrane 2018, NICE 2023 | 문헌과 가이드라인 해석 차이를 임상 맥락에서 정리 필요 |
| 기록/관찰/피드백 | WHO 2022, Maggioni et al. 2016 | UI에서 환자에게 보여줄 수 있는 지표 범위 확인 필요 |
| 안전 문구 | WHO 재활 정의, 의료기기법 제24조 | 환자용 문구로 오해가 없는지 확인 필요 |

## 재활 전문가 확인 전 보류 문장

- "기능 향상을 돕습니다."
- "회복을 지원합니다."
- "맞춤형 재활을 제공합니다."
- "환자 상태에 맞춰 자동으로 훈련합니다."
- "보행/상지 기능 개선 효과가 있습니다."
