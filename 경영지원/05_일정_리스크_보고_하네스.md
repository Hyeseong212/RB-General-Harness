# 일정 리스크 보고 하네스

## 목적

회의록, 과제 일정, 구매 일정, 외주 일정에서 마감일과 병목을 뽑아 PM과 경영지원이 볼 수 있는 리스크 보고표를 만든다.

## 입력 자료

- 회의록
- 과제 일정표
- 구매/납품 일정
- 외주 계약 일정
- 제출 마감일
- 담당자 목록

## 에이전트 역할

- 날짜, 담당자, 산출물, 의존성을 추출한다.
- 늦어지면 영향을 받는 후속 업무를 정리한다.
- 병목과 승인 대기 항목을 WARN으로 표시한다.
- 이번 주 처리 항목과 다음 주 대기 항목을 나눈다.

## 산출물 형식

```md
## 일정 요약
## 이번 주 액션
## 마감 임박 항목
## 승인 대기 항목
## 병목 리스크
## 담당자별 할 일
## PM 확인 질문
```

## 일정 리스크 표

| 마감일 | 항목 | 담당자 | 의존성 | 지연 시 영향 | 상태 |
| --- | --- | --- | --- | --- | --- |
|  |  |  |  |  | PASS/WARN/FAIL/BLOCKED |

## 검증 기준

- 날짜가 실제 문서에서 추출되었는가
- 담당자가 명확한가
- 의존성이 표시되어 있는가
- 승인 대기 항목이 따로 분리되어 있는가
- PM이 바로 확인할 질문이 남아 있는가
