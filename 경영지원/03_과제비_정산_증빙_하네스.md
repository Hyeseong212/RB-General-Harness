# 과제비 정산 증빙 하네스

## 목적

정부지원사업 또는 국가과제 관련 비용 집행 전후에 예산 항목, 증빙 자료, 정산 리스크를 정리한다.

## 입력 자료

- 사업 공고문
- 협약서 또는 사업비 집행 기준
- 예산 계획표
- 지출 예정 내역
- 견적서, 거래명세서, 세금계산서, 카드전표
- 내부 결재 문서

## 에이전트 역할

- 지출 항목을 예산 항목과 매칭한다.
- 증빙 자료 누락 여부를 확인한다.
- 정산 리스크가 있는 항목을 WARN으로 표시한다.
- 공식 공고문 또는 전담기관 문서 확인이 필요한 부분을 분리한다.
- 담당자가 확인해야 할 질문 목록을 만든다.

## 산출물 형식

```md
## 지출 요약
## 예산 항목 매칭표
## 증빙 체크리스트
## 정산 리스크
## 공식 출처 확인 필요
## 담당자 질문 목록
```

## 증빙 체크리스트

| 항목 | 필요 자료 | 보유 여부 | WARN |
| --- | --- | --- | --- |
| 견적 | 견적서 | 예/아니오 |  |
| 거래 | 거래명세서 | 예/아니오 |  |
| 세금 | 세금계산서/카드전표 | 예/아니오 |  |
| 승인 | 내부 결재 | 예/아니오 |  |
| 결과 | 납품/검수 자료 | 예/아니오 |  |

## 검증 기준

- 공고문 또는 협약서 기준이 확인되었는가
- 지출 항목이 예산 항목과 맞는가
- 증빙 자료가 지출 시점과 맞는가
- 사전 승인 또는 변경 승인 필요 항목이 표시되었는가
- 정산 불가 가능성이 있는 항목이 WARN으로 표시되었는가

## 사람 승인 게이트

과제비 집행 가능 여부, 정산 가능 여부, 예산 변경, 전담기관 제출 문구는 담당자와 PM이 확정합니다.
