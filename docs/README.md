# docs 폴더

회사 차원의 에이전트 운영 문서를 두는 폴더입니다. 부서별 하네스 자체보다는 조직 구성, 역할 분담, 발표자료에 들어갈 운영 방식 설명을 정리합니다.

## 현재 문서

- `company-role-agent-map.md`: 회사 구성원별로 에이전트를 어디에 쓰면 좋은지 정리한 문서

## 이 폴더에 넣기 좋은 문서

- 팀 운영 방식 설명
- 좌석 배정 정책
- 비용 운영 정책
- 부서별 도입 로드맵
- 발표자료용 요약 문구

## 작성 기준

문서는 바로 PPT에 옮길 수 있도록 짧은 제목, 표, 핵심 문장 중심으로 작성합니다. 세부 지침은 각 부서 폴더에 두고, 이 폴더에는 회사 전체 관점에서 보는 운영 방식을 남깁니다.
