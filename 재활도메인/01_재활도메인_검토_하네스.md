# 재활도메인 검토 하네스

## 목적

훈련 지표, 환자 문구, 발표자료 문구에서 의료적 단정 표현과 안전 관련 누락을 찾아 전문가가 검토하기 쉽게 정리한다.

## 입력 자료

- 훈련 리포트 문구
- 환자 또는 보호자 대상 화면 문구
- 운동 범위, 반복 횟수, 통증 관련 설명
- 발표자료 또는 제안서의 재활 관련 문장
- 전문가에게 확인받고 싶은 질문

## 에이전트 역할

- 의료적 단정 표현을 찾아 표시한다.
- 안전 제한과 주의 문구가 필요한 부분을 정리한다.
- 환자에게 오해될 수 있는 표현을 WARN으로 표시한다.
- 전문가에게 물어볼 질문 목록을 만든다.

## 산출물 형식

```md
## 검토 대상 요약
## 지표와 문구 분리
## 안전 제한 확인
## WARN 표현
## 수정 제안
## 전문가 질문 목록
```

## 작업 절차

1. 문구가 환자용, 전문가용, 발표용, 내부용 중 어디에 쓰이는지 분류한다.
2. 측정값, 해석, 권고, 홍보 문구를 분리한다.
3. 의료적 단정 표현을 WARN으로 표시한다.
4. 안전 제한, 중단 조건, 전문가 확인 필요 항목을 찾는다.
5. 수정 제안은 확정 문구가 아니라 검토 후보로 작성한다.

## WARN 표현 예시

| 표현 유형 | 예시 | 처리 |
| --- | --- | --- |
| 치료 효과 단정 | 회복됩니다, 치료됩니다 | 전문가 확인 필요 |
| 진단처럼 보이는 표현 | 현재 상태는 비정상입니다 | 전문가 확인 필요 |
| 안전 조건 누락 | 통증이 있어도 계속하세요 | 수정 필요 |
| 과도한 성능 표현 | 정확하게 평가합니다 | 근거 확인 필요 |

## 검증 체크리스트

| 항목 | 확인 |
| --- | --- |
| 환자용 문구와 전문가용 문구가 분리되었는가 | 예/아니오 |
| 의료적 단정 표현이 WARN으로 표시되었는가 | 예/아니오 |
| 안전 제한 또는 중단 조건이 검토되었는가 | 예/아니오 |
| 전문가 질문 목록이 남아 있는가 | 예/아니오 |
| 근거 확인이 필요한 문장이 표시되었는가 | 예/아니오 |

## 금지 사항

- 진단, 치료 효과, 처방, 의료적 권고를 확정하지 않는다.
- 임상적 안전성을 단정하지 않는다.
- 전문가 승인 없이 환자 대상 문구를 최종 확정하지 않는다.
