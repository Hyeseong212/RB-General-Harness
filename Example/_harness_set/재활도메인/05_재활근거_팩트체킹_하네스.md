# 재활근거 팩트체킹 하네스

## 목적

재활 관련 근거, 논문, 가이드라인, 식약처 자료, 의료기기 문구를 확인할 때 공식 출처와 원문 근거를 남긴다.

## 입력 자료

- 확인하려는 주장 또는 문장
- 참고 논문 또는 URL
- 식약처, 학회, 기관 가이드라인
- 발표자료 또는 제안서 문구
- 제품 기능 설명

## 에이전트 역할

- 주장을 문장 단위로 쪼갠다.
- 근거가 필요한 문장을 표시한다.
- 공식 출처 또는 원문을 확인한다.
- 발행기관, 발행연도, URL, 확인일을 남긴다.
- 적용 가능 범위를 보수적으로 정리한다.
- 전문가 검토 필요 항목을 WARN으로 표시한다.

## 산출물 형식

```md
## 확인 대상 주장
## 출처 목록
## 근거 요약
## 적용 가능 범위
## 근거 부족 항목
## 전문가 확인 필요
```

## 근거 확인표

| 주장 | 출처 | 발행기관 | 발행연도/등록일 | 적용 가능성 | 상태 |
| --- | --- | --- | --- | --- | --- |
|  |  |  |  | 높음/중간/낮음 | PASS/WARN/FAIL |

## 검증 기준

- 원문 또는 공식 출처를 확인했는가
- 출처의 발행기관과 날짜를 남겼는가
- 주장과 근거의 적용 범위가 맞는가
- 치료 효과를 과장하지 않았는가
- 전문가 검토가 필요한 항목이 표시되어 있는가

## 금지 사항

- 논문 초록만 보고 확정하지 않는다.
- 의료적 효과를 제품 성능처럼 단정하지 않는다.
- 전문가 승인 없이 환자용 문구에 반영하지 않는다.
