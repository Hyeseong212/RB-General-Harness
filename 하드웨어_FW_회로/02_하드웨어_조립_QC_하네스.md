# 하드웨어 조립 QC 하네스

## 목적

장비 조립 과정에서 부품, 케이블, 체결, 외관, 라벨, 기본 동작 확인 항목을 표준화한다.

## 입력 자료

- 조립 절차서
- 부품 목록과 BOM
- 조립 사진
- 케이블 연결표
- 장비 시리얼 또는 버전
- QC 기준

## 에이전트 역할

- 조립 단계를 순서대로 정리한다.
- 누락되기 쉬운 부품과 케이블을 체크리스트로 만든다.
- 사진 기준 확인이 필요한 항목을 표시한다.
- 장비별 시리얼, 버전, 조립일, 담당자를 기록할 양식을 만든다.
- QC 실패 시 재작업 항목을 정리한다.

## 조립 QC 체크리스트

| 단계 | 확인 항목 | 확인 방법 | 결과 | 비고 |
| --- | --- | --- | --- | --- |
| 부품 | BOM과 실제 부품 일치 | 육안 | PASS/WARN/FAIL |  |
| 케이블 | 커넥터 방향과 체결 | 육안/사진 | PASS/WARN/FAIL |  |
| 체결 | 나사/브라켓 고정 | 육안/토크 기준 | PASS/WARN/FAIL |  |
| 외관 | 파손/찍힘/오염 | 육안/사진 | PASS/WARN/FAIL |  |
| 기본동작 | 전원/LED/초기화 | 장비 테스트 | PASS/WARN/FAIL |  |

## 산출물 형식

```md
## 장비 정보
## 조립 단계
## QC 체크리스트
## 사진 필요 항목
## 실패 및 재작업 항목
## 출고 전 승인 필요
```

## 사람 승인 게이트

출고, 전원 관련 문제, 체결 불량, 센서/모터 이상, 안전 커버 누락은 하드웨어 담당자가 직접 확인합니다.
