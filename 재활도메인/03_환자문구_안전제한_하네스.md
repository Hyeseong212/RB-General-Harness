# 환자문구 안전제한 하네스

## 목적

환자 또는 보호자에게 보이는 앱 문구, 리포트 문구, 안내 문구에서 오해 가능성과 안전 제한 누락을 찾는다.

## 입력 자료

- 앱 화면 문구
- 훈련 시작/종료 안내
- 통증, 피로, 중단 관련 문구
- 리포트 결과 문구
- 보호자 또는 치료사용 안내 문구

## 에이전트 역할

- 환자에게 직접 보이는 문구를 분리한다.
- 명령형, 단정형, 불안 유발 표현을 표시한다.
- 안전 제한 또는 중단 조건이 필요한 위치를 찾는다.
- 더 부드럽고 중립적인 표현 후보를 제시한다.
- 전문가 승인 필요 항목을 WARN으로 표시한다.

## 문구 검토표

| 원문 | 문제 유형 | 위험 이유 | 수정 후보 | 승인 필요 |
| --- | --- | --- | --- | --- |
|  | 단정/명령/불안/누락 |  |  | 예/아니오 |

## 안전 제한 확인 항목

- 통증 증가 시 중단 또는 문의 안내가 있는가
- 어지러움, 호흡 불편, 급격한 피로 등 이상 반응 표현이 검토되었는가
- 환자 스스로 판단하기 어려운 표현이 있는가
- 의료진 또는 전문가에게 문의해야 할 상황이 표시되어 있는가
- "계속 진행"을 강요하는 표현이 없는가

## 산출물 형식

```md
## 검토 대상
## 환자 노출 문구
## WARN 문구
## 안전 제한 필요 위치
## 수정 후보
## 전문가 승인 필요
```

## 금지 사항

- 환자 상태별 처방 또는 운동 강도를 확정하지 않는다.
- 안전 문구를 에이전트가 최종 확정하지 않는다.
- 법적 고지 문구를 전문가 검토 없이 확정하지 않는다.
