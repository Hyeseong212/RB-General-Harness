# templates 폴더

반복 업무를 에이전트에게 맡길 때 복사해서 쓰는 기본 양식을 두는 폴더입니다.

## 현재 문서

- `task-brief.md`: 에이전트 작업 요청을 정리하는 기본 양식

## 사용 방법

1. `task-brief.md`를 기준으로 목적, 입력 자료, 산출물을 적는다.
2. 작업 성격에 맞는 부서 폴더의 README를 확인한다.
3. 에이전트에게 작업을 맡긴다.
4. 결과를 검증하고, 필요한 경우 `docs/` 또는 부서 폴더에 산출물을 남긴다.

## 양식에 꼭 들어가야 할 항목

- 작업 목적
- 입력 자료
- 제외 범위
- 원하는 산출물
- 검증 기준
- 사람 승인 필요 항목

## 추가 템플릿

- `advanced-task-brief.md`: 복잡한 작업을 맡기기 위한 고급 Task brief
- `handoff-note.md`: 작업 종료 후 다음 담당자에게 넘기는 인계 문서
- `fact-check-log.md`: 법률, 규제, 가격, 의료기기, 개인정보 등 최신성 확인이 필요한 정보의 출처 기록 양식
