# 재활도메인 폴더

재활 박사님 또는 도메인 전문가가 훈련 지표, 임상적 표현, 안전 제한, 환자 대상 문구를 검토할 때 사용하는 하네스 폴더입니다. 에이전트는 의료 판단을 대신하지 않고, 전문가가 빠르게 검토할 수 있도록 표현, 지표, 위험 항목, 질문 목록을 정리합니다.

## 재활도메인에서 에이전트가 맡기 좋은 일

- 훈련 리포트 문구에서 의료적 단정 표현을 찾는다.
- 환자 또는 보호자에게 오해될 수 있는 UI 문구를 WARN으로 표시한다.
- ROM, 반복 횟수, 수행 시간, 통증 입력 등 훈련 지표를 표로 정리한다.
- 안전 제한, 중단 조건, 주의 문구가 필요한 부분을 찾는다.
- 발표자료나 제안서에서 치료 효과를 과장하는 문장을 검토한다.
- 재활 전문가에게 물어볼 질문 목록을 만든다.

- `03_환자문구_안전제한_하네스.md`: 환자/보호자 대상 문구와 중단 조건 검토
- `04_임상자문_질문정리_하네스.md`: 재활 박사님 검토용 질문 목록 정리
- `05_재활근거_팩트체킹_하네스.md`: 논문, 가이드라인, 식약처 자료 등 근거 확인 로그

- 훈련 리포트 문구
- 앱 또는 관리자 화면 문구
- 운동 범위, 반복 횟수, 통증, 수행 시간 등 지표 설명

## 검증 기준

- 의료적 단정 표현이 표시되어 있는가
- 훈련 지표가 실제 측정 가능한 값과 해석 문구로 분리되어 있는가
- 안전 제한과 주의 문구가 빠지지 않았는가
- 환자에게 오해될 수 있는 표현이 따로 정리되어 있는가
- 전문가 검토가 필요한 문장이 명확히 표시되어 있는가
- 논문, 법령, 식약처 가이드라인이 필요한 경우 공식 출처 또는 원문 확인 로그가 있는가

## 사람 승인 게이트

진단, 치료 효과, 의료기기 성능, 환자 안전, 임상적 권고와 관련된 문구는 재활 전문가와 PM이 최종 확인합니다. 에이전트 결과는 검토 보조 자료이며 의료적 판단의 최종 근거가 아닙니다.

## 재활도메인 하네스에서 특히 조심할 점

- "치료된다", "회복된다", "효과가 입증됐다"처럼 단정하지 않는다.
- 환자 상태에 따른 금기 또는 중단 조건은 전문가 확인으로 넘긴다.
- 리포트 점수는 측정값과 해석을 분리한다.
- 논문이나 가이드라인을 인용할 때는 원문, 발행기관, 발행연도를 확인한다.
