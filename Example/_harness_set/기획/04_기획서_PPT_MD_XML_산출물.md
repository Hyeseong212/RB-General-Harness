# 기획서 PPT MD XML 산출물

## 목적

기획 결과는 말로만 남기지 않고, 재사용 가능한 산출물로 남긴다.

대표 산출물은 PPT 기획서, Markdown 문서, 표, XML 구조 초안이다.

## PPT 기획서

PPT 기획서는 Claude나 다른 문서형 에이전트를 활용하기 좋다.

입력해야 할 것:

- 발표 목적
- 대상 청중
- 핵심 메시지
- 반드시 들어갈 사례
- 슬라이드 수
- 발표 톤
- 참고 자료

출력받을 것:

- 슬라이드 목차
- 각 슬라이드 핵심 문장
- 시각화 아이디어
- 발표 대본 초안

## Markdown 문서

MD는 내부 공유, 작업 기준, 회의록, 가이드라인에 적합하다.

좋은 MD 산출물:

- 제목 구조가 명확하다.
- 표로 정리할 내용은 표로 만든다.
- 결정사항과 보류사항을 구분한다.
- 다음 작업이 보인다.

## XML 구조 초안

XML은 기계가 읽어야 하는 설정, 데이터 구조, 문서 변환 기준을 잡을 때 사용한다.

주의:

- XML은 사람이 최종 스키마를 검토해야 한다.
- 필드명, 타입, 필수 여부를 명시한다.
- 예시 데이터와 실제 운영 데이터를 섞지 않는다.

## 산출물 요청 기본형

```text
아래 내용을 [PPT/MD/XML] 형식으로 정리해줘.
목적:
대상 독자:
반드시 포함할 내용:
제외할 내용:
출력 형식:
검증 기준:
```
