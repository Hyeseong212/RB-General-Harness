# 경영지원 운영 하네스

## 목적

구매, 외주, 과제비, 일정 관리 자료를 비교 가능한 형태로 정리하고, 사람이 결정해야 할 항목을 분리한다.

## 입력 자료

- 견적서, 제안서, 계약 초안
- 구매 요청 내용
- 과제비 항목과 예산 기준
- 일정, 마감일, 제출처
- 내부 의사결정 기준

## 에이전트 역할

- 문서의 핵심 항목을 표로 정리한다.
- 금액, 조건, 일정, 리스크를 분리한다.
- 비교 기준이 다른 항목을 WARN으로 표시한다.
- 담당자가 확인해야 할 질문 목록을 만든다.

## 산출물 형식

```md
## 요약
## 비교표
## 비용 항목
## 일정 및 마감
## 리스크
## 공식 출처 확인 필요
## 승인 필요 항목
```

## 작업 절차

1. 자료 종류를 분류한다.
   - 견적/구매
   - 과제비/정산
   - 외주/계약
   - 일정/보고
2. 금액, 일정, 조건, 담당자를 표로 만든다.
3. 비교 기준이 다른 항목을 WARN으로 표시한다.
4. 공식 출처 확인이 필요한 항목을 분리한다.
5. 사람이 결정해야 할 항목과 단순 정리 항목을 나눈다.
6. 다음 담당자가 바로 볼 수 있는 인계 메모를 남긴다.

## 검증 체크리스트

| 항목 | 확인 |
| --- | --- |
| 금액과 VAT가 분리되어 있는가 | 예/아니오 |
| 수량, 단가, 총액이 맞는가 | 예/아니오 |
| 납기, 보증, 유지보수 조건이 있는가 | 예/아니오 |
| 계약 또는 과제비 기준 원문을 확인했는가 | 예/아니오 |
| 승인권자가 필요한 항목이 표시되어 있는가 | 예/아니오 |
| 외부 제출 전 검토 필요 문구가 표시되어 있는가 | 예/아니오 |

## 금지 사항

- 최종 구매, 계약, 결제 결정을 대신하지 않는다.
- 법적 해석이나 세무 판단을 확정하지 않는다.
- 외부 제출용 문구를 사람 검토 없이 확정하지 않는다.
