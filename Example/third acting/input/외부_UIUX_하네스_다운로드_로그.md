## 이번 PPT에 반영할 항목

- 화면ID
- 사용자 역할
- 진입 조건
- 노출 데이터
- 입력 필드
- 버튼/액션
- 상태 전이
- 저장 이벤트
- 예외 케이스
- 완료 조건
- 전문가 확인 항목

## 이번 PPT에서 제거할 항목

- 경험을 바꾼다
- 안심을 준다
- 판단을 돕는다
- 흐름을 본다
- 방향을 제시한다
- 재미를 만든다
- 참여를 높인다

위 표현은 발표용 문장으로는 가능하지만, 실무 개발 인계 문서에서는 화면·데이터·상태 단어로 바꾼다.
