# 정리 문서화 에이전트

## 목적

정리 문서화 에이전트는 흩어진 내용을 같은 맥락으로 묶고, 다음 사람이 읽을 수 있는 문서로 바꾸는 역할이다.

## 적용 업무

- 회의록 정리
- 액션 아이템 추출
- 요구사항 목록화
- 리스크 정리
- 보고서 초안 작성
- 진행 상황 요약
- 부서 간 handoff 문서 작성

## 좋은 입력

- 원본 회의록
- 참고 자료
- 문서 목적
- 대상 독자
- 원하는 출력 형식
- 제외할 내용

## 출력 예시

### 회의록 정리

```text
결정사항:
- 

액션 아이템:
- 담당자 / 할 일 / 기한

보류사항:
- 

리스크:
- 
```

### 요구사항 정리

```text
요구사항:
근거:
담당 부서:
입력 자료:
출력 산출물:
검증 기준:
```

## 검증 기준

- 중복 내용이 제거됐는가
- 결정사항과 의견이 구분됐는가
- 담당자와 기한이 보이는가
- 다음 작업으로 넘길 수 있는가
- 추측과 사실이 구분됐는가
