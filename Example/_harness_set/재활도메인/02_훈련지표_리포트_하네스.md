# 훈련지표 리포트 하네스

## 목적

앱 또는 관리자 화면에서 보여주는 훈련 지표를 측정값, 해석, 주의 문구로 분리해 재활 전문가가 검토하기 쉬운 형태로 정리한다.

## 입력 자료

- 훈련 리포트 화면
- 측정 지표 목록
- 데이터 컬럼 설명
- 환자 또는 보호자에게 보여줄 문구
- 전문가용 상세 리포트 문구
- 기존 발표자료 또는 제안서 문구

## 에이전트 역할

- 지표 이름과 의미를 표로 정리한다.
- 측정값과 해석 문구를 분리한다.
- 환자에게 보이는 문구와 전문가용 문구를 나눈다.
- 근거 확인이 필요한 해석을 WARN으로 표시한다.
- 재활 전문가에게 확인할 질문을 만든다.

## 지표 정리표

| 지표 | 측정값 | 단위 | 환자용 표현 | 전문가용 표현 | WARN |
| --- | --- | --- | --- | --- | --- |
| ROM |  | 도/범위 |  |  |  |
| 반복 횟수 |  | 회 |  |  |  |
| 수행 시간 |  | 초/분 |  |  |  |
| 통증 입력 |  | 점수 |  |  |  |
| 좌우 차이 |  | 비율/차이 |  |  |  |

## 산출물 형식

```md
## 리포트 목적
## 지표 정리표
## 환자용 문구
## 전문가용 문구
## 해석 주의사항
## 전문가 확인 질문
```

## 검증 기준

- 지표가 실제 수집 가능한 데이터와 연결되어 있는가
- 단위가 명확한가
- 해석 문구가 측정값보다 과장되어 있지 않은가
- 환자에게 불안감을 줄 수 있는 표현이 WARN으로 표시되었는가
- 전문가 확인이 필요한 문장이 따로 정리되었는가

## 사람 승인 게이트

훈련 지표의 임상적 의미, 환자용 해석 문구, 치료 효과와 관련된 표현은 재활 전문가가 최종 확인합니다.
