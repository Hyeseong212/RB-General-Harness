# 임상자문 질문정리 하네스

## 목적

재활 박사님 또는 외부 전문가에게 자문을 요청하기 전에 질문을 구조화하고, 답변을 받아야 할 우선순위를 정리한다.

## 입력 자료

- 제품 기능 설명
- 훈련 지표 설명
- 리포트 문구
- 사용자 시나리오
- 현재 고민 중인 문장 또는 기능
- 발표자료 또는 제안서 문구

## 에이전트 역할

- 질문을 안전, 지표, 문구, 임상 근거, 사용성으로 분류한다.
- 답변이 없으면 진행이 어려운 질문을 우선순위 높음으로 표시한다.
- 단순 의견 질문과 의사결정 질문을 분리한다.
- 전문가 답변을 기록할 표를 만든다.

## 질문 분류표

| 분류 | 질문 | 필요한 이유 | 우선순위 | 답변자 | 상태 |
| --- | --- | --- | --- | --- | --- |
| 안전 |  |  | 높음/중간/낮음 |  | 대기 |
| 지표 |  |  | 높음/중간/낮음 |  | 대기 |
| 문구 |  |  | 높음/중간/낮음 |  | 대기 |
| 근거 |  |  | 높음/중간/낮음 |  | 대기 |

## 산출물 형식

```md
## 자문 목적
## 제품/기능 요약
## 질문 분류표
## 우선 확인 질문
## 자문 후 반영 위치
## 남은 리스크
```

## 검증 기준

- 질문이 너무 추상적이지 않은가
- 전문가가 답하기 위해 필요한 배경이 포함되어 있는가
- 답변 후 반영할 파일 또는 화면이 표시되어 있는가
- 외부 자문 전 민감 정보가 제거되었는가
