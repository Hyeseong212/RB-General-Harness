# 디자인 폴더

디자인, UI/UX, PPT 시각자료, 캡처 이미지 정리에 사용하는 하네스 폴더입니다. 에이전트는 최종 디자인 의사결정을 대신하지 않고, 발표자료와 화면 검토에 필요한 재료를 정리하는 역할을 맡습니다.

## 이 폴더를 쓰는 경우

- PPT에 들어갈 캡처 이미지를 선별하고 순서를 잡을 때
- UI/UX 개선 포인트를 화면 단위로 정리할 때
- 디자인 레퍼런스와 현재 산출물을 비교할 때
- 발표자료의 페이지별 시각 메시지를 정리할 때
- 동일한 스타일로 캡처 카드, 표, 설명 문구를 맞출 때

## 문서 구성

- `02_UI_UX_디자인_가이드라인.md`: 화면과 사용자 흐름 검토 기준

## 검증 기준

- 캡처 이미지가 발표 메시지와 직접 연결되는가
- 민감 정보가 마스킹되어 있는가
- 한 페이지에 들어가는 문구가 너무 많지 않은가
- 디자인이 기존 PPT 스타일과 어긋나지 않는가
- 이미지, 텍스트, 표가 겹치지 않는가

## 사람 승인 게이트

대외 발표용 이미지, 환자 또는 계정이 노출될 수 있는 캡처, 제품 성능을 암시하는 화면은 디자인 담당자와 PM이 최종 확인합니다.
