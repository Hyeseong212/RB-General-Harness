# second acting

이 폴더는 2막 실행을 위한 빈 작업공간입니다.

## 목표

1막 결과를 PPT 디자인 가이드라인과 엮어 발표자료 산출물로 변환합니다.

## 사용할 하네스

- `_harness_set/templates/advanced-task-brief.md`
- `_harness_set/templates/handoff-note.md`
- `_harness_set/디자인/02_UI_UX_디자인_가이드라인.md`
- `_harness_set/디자인/03_PPT_디자인_가이드라인.md`
- `_harness_set/디자인/04_PPT_디자인_레퍼런스_사이트.md`
- `_harness_set/디자인/05_캡처_QA_하네스.md`

## 폴더 사용법

- `input`: 1막 `output` 결과와 참고 디자인 자료를 넣는 곳
- `output`: 슬라이드 구성안, PPT 문구, PPTX 산출물, 렌더링 검증 결과를 남기는 곳

## 실행 시 주의

- 긴 조사 내용을 그대로 장표에 넣지 않는다.
- 한 장에는 하나의 메시지만 둔다.
- 실제 화면이 없으면 캡처 자리 또는 UI 예시 자리로 표시한다.
- 재활의학 관련 주장은 전문가 검토 항목으로 분리한다.
