# 원본 복사 목록

이 폴더의 하네스 파일은 상위 폴더에서 가져온 원본 파일명과 내용을 그대로 유지합니다.

영어명으로 바꾸거나 요약본으로 바꾸지 않고, 다음 쓰레드에서 “어떤 원본 하네스를 가져왔는지” 바로 확인할 수 있게 세팅했습니다.

## 복사 기준

- 폴더명 유지
- 파일명 유지
- 본문 내용 유지
- 실행 결과물 생성 없음
