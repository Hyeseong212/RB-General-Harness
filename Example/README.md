# Example: 하네스 실행 전 세팅 폴더

이 폴더는 실제 Act를 실행하기 전, 다음 쓰레드에서 바로 사용할 하네스를 모아두는 준비 공간입니다.

여기서는 브레인스토밍, 자료조사, PPT 제작, UI/UX 설계를 실행하지 않습니다. 실행은 별도 쓰레드에서 진행합니다.

## 사용 시나리오

예시 주제:

- 이승로봇 콘텐츠 기획

진행 방식:

1. `first acting`에서 브레인스토밍, 자료조사, 재활의학 관점을 묶어 콘텐츠 방향을 잡는다.
2. `second acting`에서 1막 결과와 PPT 디자인 하네스를 엮어 PPT 산출물로 바꾼다.
3. UI/UX까지 확장할 경우 별도 Act를 추가한다.

## 폴더 구성

| 폴더 | 역할 |
| --- | --- |
| `_harness_set` | 이번 예제에 사용할 기존 하네스 모음 |
| `first acting` | 1막 실행 전 입력/출력 위치만 잡아둔 폴더 |
| `second acting` | 2막 실행 전 입력/출력 위치만 잡아둔 폴더 |

## 운영 기준

- 이 폴더는 실행 결과물이 아니라 준비물 세트다.
- 각 Act는 별도 쓰레드에서 시작한다.
- 이전 Act의 `output`을 다음 Act의 `input`으로 넘긴다.
- 재활의학, 치료 효과, 안전 문구는 에이전트 결과만으로 확정하지 않고 전문가 검토로 넘긴다.
