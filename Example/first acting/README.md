# first acting

이 폴더는 1막 실행을 위한 빈 작업공간입니다.

## 목표

브레인스토밍, 자료조사, 재활의학 관점을 묶어 이승로봇 콘텐츠 방향을 정리합니다.

## 사용할 하네스

- `_harness_set/templates/advanced-task-brief.md`
- `_harness_set/공통/03_검증_게이트_체크리스트.md`
- `_harness_set/공통/08_팩트체킹_법률_규제_하네스.md`
- `_harness_set/기획/02_브레인스토밍_에이전트.md`
- `_harness_set/기획/05_리서치_의사결정_하네스.md`
- `_harness_set/재활도메인/01_재활도메인_검토_하네스.md`
- `_harness_set/재활도메인/04_임상자문_질문정리_하네스.md`
- `_harness_set/재활도메인/05_재활근거_팩트체킹_하네스.md`

## 폴더 사용법

- `input`: 제품 설명, 기존 회의록, 참고자료를 넣는 곳
- `output`: 브레인스토밍 결과, 자료조사 로그, 재활 검토 질문을 남기는 곳

## 실행 시 주의

- 치료 효과를 단정하지 않는다.
- 자료조사가 필요한 문장은 공식 출처 또는 논문 원문 확인 로그를 남긴다.
- 재활 전문가 확인이 필요한 문장은 별도 목록으로 분리한다.
