- 작업 범위: 신규 기획 PPT 구성안·문구 초안 작성, PPTX 생성, 캡처 기반 디자인/맥락 QA.

## 정정 히스토리

- 기존 작성 방향: 기존 하네싱 발표자료의 9페이지 이후에 삽입하는 장표 구성안
- 정정된 방향: 별도의 신규 "이승로봇 게임재활 콘텐츠 기획 PPT"를 만드는 구성안
- 이유: second acting에서 말하는 PPT는 기존 발표자료 보강용이 아니라, first acting 결과를 바탕으로 새 콘텐츠 기획안을 만드는 산출물이다.
- 추가 확인: 기존 발표자료 9페이지 이후에 넣는 작업은 나중에 할 일로 보류한다. 현재는 히스토리만 남긴다.

## 완료 요약

first acting의 자료조사/브레인스토밍 결과를 바탕으로, 신규 "이승로봇 게임재활 콘텐츠 기획 PPT"의 흐름을 정리하고 PPTX로 출력했다. 중심 메시지는 "간단한 재활운동을 반복 숙제가 아니라, 목표·반응·기록이 있는 짧은 게임 루프로 설계한다"이다.

## 권장 장표 수

신규 기획 PPT는 10장 구성을 권장한다.

1. 운동은 그대로인데, 경험이 달라진다
2. 기존 재활운동 콘텐츠는 반복의 이유를 보여주기 어렵다
3. 게임화의 핵심은 재미가 아니라 반복 루프를 만드는 것이다
4. 기본 운동은 작은 게임 미션으로 바뀐다
5. 환자에게는 오늘의 미션을 보여준다
6. 치료사에게는 게임 결과가 관찰 기록이 된다
7. 보호자와 관리자는 안전과 운영 흐름을 본다
8. 재미있는 문장과 안전한 문장은 따로 검토한다
9. first acting과 second acting은 역할이 다르다
10. 다음 단계는 화면 흐름과 작은 프로토타입이다

## 꼭 유지할 방향

- 신규 기획 PPT로 본다.

- 게임화 아이디어가 주인공이고, 팩트체킹은 안전장치다.
- 환자/치료사/보호자/관리자 관점을 분리한다.
- 치료 효과, 회복, 기능 개선 표현은 사용하지 않는다.
