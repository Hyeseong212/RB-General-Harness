# second acting 작업 히스토리

- 작성일: 2026-05-12
- 현재 단계: second acting
- 현재 목적: first acting 결과를 바탕으로 신규 "이승로봇 게임재활 콘텐츠 기획 PPT"의 구성안과 장표 문구를 정리한다.

## 현재 확정된 방향

second acting에서 만드는 PPT는 기존 하네싱 발표자료의 9페이지 이후에 삽입하는 장표가 아니다.

현재 second acting은 별도의 신규 기획 PPT를 준비하는 단계다.  
주제는 "간단한 재활운동을 게임운동 콘텐츠로 승화시키는 기획"이다.

## 보류된 작업

기존 하네싱 발표자료의 9페이지 이후에 내용을 넣는 작업은 나중에 할 일로 보류한다.

보류 이유:

- 현재 second acting의 목적은 기존 발표자료 보강이 아니라 신규 기획 PPT 구성이다.
- 9페이지 이후 삽입 작업은 별도 PPT 편집 단계에서 다룬다.
- 기존 발표자료에 넣을 경우, 신규 기획 PPT의 내용을 그대로 넣지 않고 "운영 예시"로 다시 압축해야 한다.

## 나중에 할 일

기존 하네싱 발표자료 9페이지 이후 삽입 작업을 다시 진행할 때는 아래 기준으로 본다.

1. 신규 기획 PPT 전체를 넣지 않는다.
2. "자료조사 에이전트 세팅 운영 예시"만 2~3장으로 압축한다.
3. first acting과 second acting의 역할 분리를 보여준다.
4. 게임재활 콘텐츠 자체보다, 하네싱 방식이 기획 업무에도 적용된다는 점을 보여준다.
5. 기존 비용/운영 장표와 자연스럽게 이어지도록 별도 연결 문구를 만든다.
