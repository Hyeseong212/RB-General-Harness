# 경영지원 폴더

경영지원, 구매, 외주 비교, 과제비, 일정 관리 업무에 사용하는 하네스 폴더입니다. 에이전트는 비용 결정을 대신하지 않고, 결정을 위한 비교표, 증빙 체크, 질문 목록, 리스크 메모를 빠르게 정리하는 역할을 맡습니다.

## 경영지원에서 에이전트가 맡기 좋은 일

- 견적서 2~3개를 같은 기준으로 비교한다.
- 구매 요청서에서 누락된 수량, 단가, VAT, 납기, 보증 조건을 찾는다.
- 과제비 집행 전 증빙 자료와 예산 항목을 대조한다.
- 외주 제안서의 범위, 산출물, 검수 기준, 지식재산권 조항을 질문 목록으로 바꾼다.
- 제출 마감일, 계약일, 결제일, 보고일을 일정 리스크 표로 정리한다.
- 회의 후 경영지원 담당자가 처리해야 할 액션 아이템을 뽑는다.

## 문서 구성

- `03_과제비_정산_증빙_하네스.md`: 과제비 집행, 정산, 증빙 누락 점검
- `04_외주_계약_검토_하네스.md`: 외주 제안서, 계약 초안, 검수 조건 점검
- `05_일정_리스크_보고_하네스.md`: 마감일, 결제일, 보고일, 병목 리스크 정리

## 검증 기준

- 금액, 수량, 단가, VAT 포함 여부가 분리되어 있는가
- 비교 기준이 동일한가
- 결정이 필요한 항목과 단순 정리 항목이 구분되어 있는가
- 공고문, 계약서, 가격 정책처럼 최신성이 필요한 내용은 공식 출처 확인이 표시되어 있는가
- 계약, 결제, 외부 제출 전에 사람 승인 항목이 표시되어 있는가

## 사람 승인 게이트

최종 구매 결정, 계약 조건, 입금 또는 결제, 과제비 집행, 외주 발주, 법률/세무 판단은 담당자와 PM 또는 대표 승인 후 진행합니다.

## 경영지원 하네스에서 특히 조심할 점

- 에이전트가 "싸다", "적합하다", "계약 가능하다"를 확정하지 않게 한다.
- 견적 조건이 다른 항목은 같은 표에 넣되 WARN으로 표시한다.
- 정부지원사업, 과제비, 정산 기준은 공고문 또는 전담기관 문서 원문 확인을 요구한다.
- 계약 조항은 요약까지만 맡기고 최종 판단은 사람 승인으로 넘긴다.
