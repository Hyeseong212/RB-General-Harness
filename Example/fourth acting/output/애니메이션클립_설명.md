# 애니메이션 클립 샘플 설명

- 작성일: 2026-05-12
- 단계: fourth acting
- 기준 해상도: 모바일 세로 `1080x1920`
- 목적: 연결된 화면이 실제로 어떻게 넘어가는지, 어떤 컴포넌트 액션이 화면 전환을 만드는지 보여주는 샘플 예시를 만든다.

## 생성 파일

| 파일 | 용도 |
| --- | --- |
| `이승로봇_연결화면_애니메이션클립.svg` | SVG/XML 기반 18초 루프 애니메이션 클립 |
| `이승로봇_연결화면_애니메이션클립_preview.html` | 브라우저에서 SVG를 크게 확인하는 프리뷰 |
| `animation_clip_timeline.xml` | 화면ID, 구간, 이벤트, 다음 화면을 적은 XML 타임라인 명세 |
| `애니메이션클립_설명.md` | 산출물 설명 |
| `fourth_acting_작업히스토리.md` | 작업 히스토리 |

## 클립 구성

| 시간 | 화면 | 액션 | 다음 |
| --- | --- | --- | --- |
| 0~3초 | P-01 운동 선택 | 운동 카드 탭 | P-02 |
| 3~6초 | P-02 미션 안내 | 시작 버튼 탭 | P-03 |
| 6~10초 | P-03 미션 수행 | 쉬어가기 탭 | P-04A |
| 10~13초 | P-04A 쉬어가기 | 계속하기 탭 | P-03 |
| 13~15초 | P-03 미션 수행 복귀 | 시간 종료 | P-05 |
| 15~18초 | P-05 완료 요약 | 저장하고 나가기 | 종료 |

## 사용하는 방법

1. `이승로봇_연결화면_애니메이션클립_preview.html`을 브라우저에서 연다.
2. 또는 `이승로봇_연결화면_애니메이션클립.svg`를 브라우저에 직접 연다.
3. 애니메이션은 18초 루프로 반복된다.

## 주의사항

- 이 클립은 개발 인계용 샘플이며 최종 UI 애니메이션 디자인이 아니다.
- 치료, 회복, 효과 입증처럼 보일 수 있는 문구는 넣지 않았다.
- 실제 센서 반응, 중단 기준, 환자 피드백 문구는 재활 전문가 확인 후 확정해야 한다.
- fifth acting에는 데이터 플로우 차트 샘플을 별도로 추가하는 흐름으로 남긴다.
